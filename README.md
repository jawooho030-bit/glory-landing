# 🇰🇷 GLORY 明洞店 Landing Page（Meta Ads 導流專用）

本專案為 GLORY 韓國皮膚管理中心之 **Meta / Instagram 廣告轉換落地頁（Landing Page）**  
核心目標：**將廣告流量轉換為 LINE 諮詢客資**

---

## 🎯 核心轉換目標（Primary KPI）

本頁所有設計圍繞以下目標：

- LINE 點擊率（CTR → LINE Click）
- LINE 諮詢轉換率（Lead Conversion）
- 廣告 CPA 降低（Cost per Lead）
- 停留時間提升（Engagement）

---

## 💰 流量來源（Traffic Source）

本頁主要接收流量：

- Meta Ads（Instagram / Facebook）
- Reels / Story 廣告
- KOL / Influencer 連結
- 小紅書導流（可選）

---
