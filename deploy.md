# 🚀 GLORY Landing Page 部署指南（GitHub 版）

本專案使用 GitHub Pages 進行靜態網站部署。

適用於：
- Meta / IG 廣告導流頁
- LINE 諮詢 Landing Page
- 無伺服器靜態網站

---

## 🌐 1. 部署方式（GitHub Pages）

### 📌 Step 1：進入 Repository

進入：glory-landing

---


### 📌 Step 2：進入 Settings

點：Settings → Pages


---

### 📌 Step 3：設定 Source

在 Pages 設定中選：

- Source：Deploy from a branch
- Branch：`main`
- Folder：`/ (root)`

然後按 **Save**

---

### 📌 Step 4：取得網址

GitHub 會自動產生網址：
